# Что это
Это мини шаблон для MSA. 

# Предостережение
Цель проекта - пошупать микросервисы, реализовать аутентификацию и освоить API Gateway. Так что использовать его для прода или как полноценнейший шаблон не стоит (как минимум пока что). Если все-таки решитесь, то перегенерируйте ключи. 

# Запуск
Используйте Docker.

# Технологии проекта
- Kong API Gateway
- Kafka
- FastAPI
- SQLAlchemy
- PyJWT
- Dishka
- Docker / docker compose
- чуть-чуть lua
